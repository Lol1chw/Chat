# Chat App

A simple application "chat" on vue

Tech stack: 
* Typescript
* Vue
* Pinia
* VueUse
* Vue-Router


## Project Setup
Locally:
```
git clone https://github.com/Lol1chw/Chat.git

pnpm install
```
Deployed: [Chat-app](https://chat-vue-ebon.vercel.app/)

# Project Run
Dev mode: 
```
pnpm dev
```

Build:
```
1. pnpm build
2. pnpm preview
```
